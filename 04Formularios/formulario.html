Todos los formularios que se manejan en la web utilizan métodos para el envío o respuesta de la información. Los más comunes son:


GET: Este método se encarga de enviar la información por la URL o salida estándar del navegador, esto propicia que se pueda visualizar toda la información del formulario.
POST: Este método se encarga de enviar la información por la URL pero de forma oculta, lo que ofrece cierto nivel de seguridad. Este método se usa comúnmente para insertar un nuevo registro dentro de los sistemas.
